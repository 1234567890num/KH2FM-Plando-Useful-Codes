# General Drive Gauge Codes

```
// Drive Bar Percentage Filled
patch=1,EE,01C6C900,extended,000000AA // Values more than 100 will carry over to the next gauge(s).

// Current Drive Gauge
patch=1,EE,01C6C901,extended,000000BB // You can enter more than your current max and it'll work just fine.
// Values 0-9 are shown as intended.
// Values 10-19 (A-13 in hex) only have their ones digit shown and in a different color.
// Values above 19 (0x13) don't have their digits shown but still works just fine.
// Putting a value above max will work just fine, but the game won't let you increase it normally (hitting, getting hit, drive recoveries).
// Also the game will restore it to normal max when it normally recovers all drive (going to world map, etc).

// Max Drive Gauge
patch=1,EE,01C6C902,extended,000000CC // You can put in more than 9 but with the display glitches above.

// Remaining Form/Summon Time
patch=1,EE,21C6C904,extended,EEEEEEEE // A float in frames (60FPS). If it's below max time then it'll drop to whatever the max time is.

// Max Form/Summon Time
patch=1,EE,21C6C908,extended,FFFFFFFF // A float in frames (60FPS). Each gauge is 600 frames long. Does not affect the gauge number shown.

// Current Form
patch=1,EE,0032F054,extended,000000GG // Takes effect in room transitions. Require the current & max time codes to work properly.
// 00 = Base, 01 = Valor, 02 = Wisdom, 03 = Limit, 04 = Master, 05 = Final. All can be dismissed normally.
// 06 = Anti. Can't do RC/be dismissed/use items, but still works in cutscenes.
// 07 = Mickey. Can't use Pearl/be dismissed. May cause a softlock.
// Higher values cause a softlock (as far as I tested).

// Current Summon
patch=1,EE,0032F055,extended,000000HH // Takes effect in room transitions (mostly). Require the current & max time codes to work properly.
// 00 = None. Causes softlock if Sora tries to summon with this in effect.
// 01 = Chicken, 02 = Genie, 03 = Stitch, 04 = Pan. When Sora is solo, it doesn't appear, can't be dismissed, and causes softlock if Sora dies. Otherwise works just fine.
// Higher values doesn't have anything appear. It can be dismissed, but the party members still won't appear. You can also drive, but can't revert.
// Combining this and current form code can cause problems when reverting. There's no problem if you revert them both simultaneously using another code though.

// Genie's Form
patch=1,EE,0032F057,extended,000000JJ // Takes effect in room transitions.
// 01 = Valor, 02 = Wisdom, 04 = Master, 05 = Final. All can be changed normally.
// Other values have Genie appear in his Valor Form but the Limit isn't available. You can still change his form though.

// Maximum Form/Summon Gauge
patch=1,EE,0032F05B,extended,000000KK // Purely cosmetic. Independent of Form/Summon Duration
```
